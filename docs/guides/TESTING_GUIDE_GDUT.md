## 🐛 已知问题和解决方案

### 问题 1: SPM 依赖解析失败
**症状**: Xcode 无法构建，提示 swift-syntax 错误
**解决方案**:
1. 关闭 Xcode
2. 删除 `~/Library/Developer/Xcode/DerivedData/FunnyPixelsApp-*`
3. 重新打开 Xcode
4. File → Packages → Reset Package Caches
5. 重新构建（Cmd+B）

### 问题 2: 后端连接失败
**症状**: API 请求超时或失败
**解决方案**:
1. 检查后端服务是否运行
2. 检查 iOS 应用的 API 配置
3. 确保真机和开发机在同一 WiFi 网络
4. 检查防火墙设置

### 问题 3: GPS 定位不准确
**症状**: 像素绘制位置偏移
**解决方案**:
1. 确保 GPS 定位权限已授予
2. 在室外测试（室内 GPS 信号弱）
3. 等待 GPS 信号稳定（通常需要10-30秒）
