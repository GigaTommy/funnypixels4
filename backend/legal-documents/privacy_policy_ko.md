# FunnyPixels 개인정보 처리방침

**시행일:** 2026년 2월 12일
**버전:** 2.0

---

## 소개

FunnyPixels(이하 "당사")는 사용자 개인정보 보호를 매우 중요시합니다. 본 개인정보 처리방침은 당사가 개인정보를 어떻게 수집, 이용, 보관, 공유, 보호하는지 설명합니다.

## 1. 수집하는 정보

### 1.1 사용자가 제공하는 정보
- **계정 정보:** 사용자명, 이메일 주소, 전화번호
- **프로필 정보:** 아바타, 표시 이름, 자기소개
- **비밀번호:** 업계 표준 암호화(bcrypt)로 저장

### 1.2 자동으로 수집되는 정보
- **위치 정보:** GPS 위치 데이터(명시적 동의 후에만 수집)
- **기기 정보:** 기기 모델, OS 버전, 고유 기기 식별자
- **사용 로그:** 앱 사용 시간, 기능 사용 빈도, 오류 로그
- **그리기 데이터:** 그린 픽셀의 좌표, 색상, 타임스탬프

### 1.3 수집하지 않는 정보
- 연락처, SMS, 통화 기록, 사진 라이브러리(프로필 사진 업로드 제외), 지속적 백그라운드 위치 추적, 생체 인식 데이터

## 2. 정보 이용 목적

| 목적 | 법적 근거 |
|------|-----------|
| 핵심 서비스 제공 | 계약 이행 |
| 계정 인증 | 계약 이행 |
| 보안 및 부정 방지 | 정당한 이익 |
| 서비스 개선 및 분석 | 정당한 이익 |
| 중요 서비스 알림 발송 | 계약 이행 |
| 법적 요건 준수 | 법적 의무 |

당사는 개인정보를 제3자에게 **판매하지 않습니다**.

## 3. 정보의 공유

3.1 원칙적으로 제3자에게 개인정보를 판매, 임대, 교환하지 않습니다.

3.2 예외: 명시적 동의, 법적 요청, 권리 보호, 데이터 보호 계약에 따른 서비스 제공업체와의 협력

## 4. 데이터 보관 및 보안

### 4.1 보관 기간
- **계정 정보:** 계정 존속 기간 및 삭제 후 30일
- **그리기 데이터:** 공개 지도의 일부로 지속 보관
- **사용 로그:** 최대 12개월
- **보안 로그:** 최대 6개월

### 4.2 보안 조치
- 전송 암호화(TLS/HTTPS), 저장 암호화(AES-256), 비밀번호 해싱(bcrypt), 정기 보안 감사

## 5. 사용자의 권리

### 5.1 모든 사용자의 권리
접근권, 정정권, 삭제권, 동의 철회권, 계정 삭제권

### 5.2 대한민국 거주자의 추가 권리
한국의 개인정보 보호법(PIPA)에 따라, 사용자는 개인정보 처리에 관한 자기결정권을 행사할 수 있으며, 개인정보의 열람, 정정·삭제, 처리정지를 요구할 수 있습니다.

### 5.3 EU/EEA 사용자의 추가 권리 (GDPR)
데이터 이동권, 처리 제한권, 이의제기권

### 5.4 권리 행사
- 이메일: privacy@funnypixels.com
- 앱 내: 설정 → 개인정보 → 데이터 요청

**30일 이내** 에 회신드리겠습니다.

## 6. 위치 정보

6.1 위치 정보는 명시적 동의 하에 그리기 기능 사용 시에만 수집합니다.

6.2 기기 설정에서 언제든 위치 권한을 변경할 수 있습니다.

본 앱은 만 13세 미만(EU/EEA에서는 16세 미만)의 아동을 대상으로 하지 않습니다.

중요한 변경 사항은 앱 내 알림을 통해 안내됩니다. 변경은 게시 후 **15일** 후에 적용됩니다.
