## 📝 文档命名规范

- 使用大写字母和下划线：`FEATURE_NAME_TYPE.md`
- 类型后缀：
  - `_GUIDE.md` - 指南
  - `_FIX.md` - 修复记录
  - `_SUMMARY.md` - 总结
  - `_PLAN.md` - 计划
  - `_COMPLETE.md` - 完成报告
