## 🎯 核心原则

### 必须满足的性能要求

| 指标 | 最低要求 | 理想目标 | 拒绝阈值 |
|-----|---------|---------|---------|
| **延迟** | <5ms | <1ms | >10ms ❌ |
| **CPU占用** | <2% | <0.5% | >3% ❌ |
| **内存增加** | <200KB | <100KB | >500KB ❌ |
| **电池影响** | <5%/小时 | <2%/小时 | >8%/小时 ❌ |
| **主线程阻塞** | <16ms | 0ms | >16ms ❌ |

### 实施检查清单

**每个反馈添加前必须回答**:
1. ❓ 延迟 <1ms? → 是 ✅ / 否 ❌ 放弃
2. ❓ CPU <0.5%? → 是 ✅ / 否 ❌ 放弃
3. ❓ 不阻塞主线程? → 是 ✅ / 否 ❌ 放弃
4. ❓ 用户真的需要? → 是 ✅ / 否 ❌ 放弃
5. ❓ 高频操作已节流? → 是 ✅ / 否 ❌ 放弃

## 🔍 性能监控代码

### 添加性能日志 (仅Debug)

```swift
#if DEBUG
extension SoundManager {
    private var performanceLog: [String: (count: Int, totalTime: TimeInterval)] = [:]

    func playWithPerformanceLog(_ effect: SoundEffect) {
        let start = CFAbsoluteTimeGetCurrent()

        play(effect)

        let duration = CFAbsoluteTimeGetCurrent() - start
        performanceLog[effect.rawValue, default: (0, 0)].count += 1
        performanceLog[effect.rawValue, default: (0, 0)].totalTime += duration

        // 每100次播放报告一次
        if performanceLog[effect.rawValue]!.count % 100 == 0 {
            let avg = performanceLog[effect.rawValue]!.totalTime / Double(performanceLog[effect.rawValue]!.count)
            Logger.debug("🎵 \(effect.rawValue): 平均延迟 \(avg * 1000)ms")
        }
    }
}
#endif
```

### 性能告警

```swift
#if DEBUG
if duration > 0.005 {  // 5ms
    Logger.warning("⚠️ 音效延迟过高: \(duration * 1000)ms")
}
#endif
```
