# 地图漂流瓶标记完整实现方案

## 概述

在地图上显示附近的漂流瓶位置，使用自定义SVG图标，支持点击交互。

## 实现架构

```
┌─────────────────────────────────────────────┐
│         MapTabContent (地图主视图)           │
│  ┌───────────────────────────────────────┐  │
│  │        MapLibre地图                   │  │
│  │  ┌─────────────────────────────────┐  │  │
│  │  │  BottleAnnotation (SVG图标)     │  │  │
│  │  │  • bottle_1 (23.1415, 113.2898) │  │  │
│  │  │  • bottle_2 (23.1420, 113.2905) │  │  │
│  │  │  • bottle_3 (...)               │  │  │
│  │  └─────────────────────────────────┘  │  │
│  └───────────────────────────────────────┘  │
└─────────────────────────────────────────────┘
         ↑
         │ 数据流
         │
┌────────┴──────────┐
│ DriftBottleManager │
│  • mapMarkers[]   │
│  • refreshMarkers()│
└────────┬──────────┘
         │
         ↓
┌────────────────────┐
│ DriftBottleAPIService│
│  GET /map-markers  │
└────────────────────┘
```
