## 🔄 动画和过渡

### 1. Tab切换动画
```swift
// 水平滑动切换
withAnimation(.easeInOut(duration: 0.3)) {
    selectedTab = newTab
}

// 淡入淡出
.transition(.opacity)
```

### 2. 双击点赞动画
```swift
// 心形放大缩小
心形初始: scale(0)
心形放大: scale(2.0) + opacity(1.0)  // 0.2s
心形缩小: scale(1.5) + opacity(0)    // 0.3s

// 数字跳动
点赞数: +1 动画 + 颜色变化（灰→红）
```

### 3. 卡片出现动画
```swift
// 从下往上滑入
.transition(.move(edge: .bottom).combined(with: .opacity))
.animation(.spring(response: 0.6, dampingFraction: 0.8))
```

### 4. 下拉刷新
```swift
// 标准下拉刷新
.refreshable {
    await viewModel.loadFeed(refresh: true)
}

// 自定义loading动画: 旋转的画笔图标
```

## 🧪 原型交互流程

### 流程1: 首次打开App

```
1. App启动
   └→ 显示Splash Screen

2. 进入主界面
   └→ 默认选中"推荐"Tab
   └→ 默认选中"推荐"筛选器

3. 自动加载内容
   └→ 显示骨架屏（0.5s）
   └→ 加载首屏20条推荐

4. 开始浏览
   └→ 用户向下滚动
   └→ 自动预加载下一页
```

### 流程2: 切换筛选器

```
1. 点击"关注"胶囊
   └→ 胶囊颜色变化（白→蓝）
   └→ 音效反馈（可选）

2. 加载关注流
   └→ 显示loading（顶部进度条）
   └→ 内容淡入（0.3s动画）

3. 如果关注列表为空
   └→ 显示空状态页
   └→ 引导用户去"探索"Tab关注创作者
```

### 流程3: 探索并关注创作者

```
1. 切换到"探索"Tab
   └→ 显示热门内容

2. 滚动到"精选创作者"区域
   └→ 横向滚动浏览

3. 点击某个创作者
   └→ 进入TA的个人主页

4. 点击"关注"按钮
   └→ 按钮变为"已关注"（绿色对勾）
   └→ Toast提示"关注成功"

5. 返回"推荐"Tab
   └→ 切换到"关注"筛选器
   └→ 可以看到该创作者的动态
```

### 流程4: 发布作品并分享

```
1. 完成绘画
   └→ 自动弹出SessionSummaryView

2. 选择"分享到动态"
   └→ 弹出ShareToFeedSheet

3. 选择"编辑分享"
   └→ 打开ShareToFeedEditView
   └→ 输入创作故事

4. 点击"发布"
   └→ 上传成功
   └→ Toast提示"分享成功"
   └→ 自动跳转到"推荐"Tab

5. 在Feed中看到自己的作品
   └→ 置顶显示（可选）
   └→ 标记"刚刚发布"
```

## 🎯 关键交互细节

### 双击点赞
```
1. 用户双击卡片中央区域
   └→ 触发点赞

2. 视觉反馈
   ├→ 心形图标从中心放大（0→2x）
   ├→ 心形图标淡出（opacity: 1→0）
   └→ 底部点赞数+1，图标变红

3. haptic反馈
   └→ UIImpactFeedbackGenerator.medium.impactOccurred()
```

### 长按菜单
```
1. 用户长按卡片1秒
   └→ 触发菜单

2. 卡片轻微缩小（0.95x）
   └→ 模糊背景（blur）

3. 底部弹出菜单
   ├→ 收藏
   ├→ 不感兴趣
   ├→ 举报
   └→ 复制链接

4. 点击任意选项
   └→ 菜单消失
   └→ 执行对应操作
```

### 滑动切换（可选）
```
// 可选功能：左右滑动切换筛选器
向左滑动 → 切换到下一个筛选器（推荐→关注）
向右滑动 → 切换到上一个筛选器（关注→推荐）

实现:
.gesture(
    DragGesture()
        .onEnded { value in
            if value.translation.width < -50 {
                // 向左滑动
                nextFilter()
            } else if value.translation.width > 50 {
                // 向右滑动
                previousFilter()
            }
        }
)
```
