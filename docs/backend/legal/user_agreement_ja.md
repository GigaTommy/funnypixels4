2.1 FunnyPixels は、現実世界の地図上でピクセルアートを描き、作成し、共有できる位置情報ベースのソーシャルアプリです。

2.2 主な機能：
- 地図上でのピクセル配置・描画
- GPS による自動描画
- アライアンスの作成・参加による共同制作
