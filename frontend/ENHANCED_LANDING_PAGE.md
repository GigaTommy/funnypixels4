## 🎨 视觉设计语言

### 配色方案
```css
/* 主色调（基于项目真实颜色） */
--primary: #5A4AF4;        /* 紫色 */
--primary-dark: #4B3EE8;   /* 深紫色 */

/* 渐变色 */
--gradient-purple: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
--gradient-pink: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
--gradient-blue: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
```

### 动画效果
- **数字滚动**：CountUp 缓动动画（2秒）
- **卡片悬浮**：Y轴上移 -5px ~ -10px
- **像素浮动**：随机位移 + 透明度变化
- **按钮交互**：Scale 1.05 + 阴影增强
- **滚动渐显**：Intersection Observer + Framer Motion

### 响应式断点
```css
sm:  640px  (平板竖屏)
md:  768px  (平板横屏)
lg:  1024px (桌面)
xl:  1280px (大屏桌面)
```

## 📞 问题排查

### 常见问题

1. **页面空白**
   - 检查控制台错误
   - 确认 `npm install` 已执行
   - 清除缓存：`rm -rf node_modules/.vite`

2. **动画卡顿**
   - 减少浮动像素数量（Hero组件）
   - 使用 Chrome DevTools Performance 分析

3. **样式错乱**
   - 确认 Tailwind CSS 配置正确
   - 检查 `index.css` 是否正确导入
