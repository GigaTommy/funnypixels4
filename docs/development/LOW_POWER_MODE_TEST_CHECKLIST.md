## 🧪 功能测试清单

### 1. 基础进入/退出流程
- [ ] **启动GPS绘制**
  - 打开app，点击GPS绘制按钮
  - 选择联盟/个人颜色
  - 确认GPS绘制模式已启动

- [ ] **5秒空闲检测**
  - 启动GPS绘制后，不触摸屏幕
  - 观察是否在5秒后自动进入低功耗模式
  - 预期：屏幕变黑，显示低功耗UI

- [ ] **首次引导Toast**
  - 第一次进入低功耗模式时
  - 观察是否显示引导Toast（"已进入专注绘制模式"）
  - 预期：Toast显示3秒后自动消失
  - 第二次进入时不应再显示

- [ ] **轻触退出**
  - 在低功耗模式下，轻触屏幕任意位置
  - 预期：立即退出低功耗模式，恢复正常UI

- [ ] **长按退出（备用）**
  - 在低功耗模式下，长按屏幕0.5秒
  - 预期：退出低功耗模式

---

### 2. UI展示测试

- [ ] **黑色背景**
  - 低功耗模式下，背景是否为纯黑色
  - 预期：完全遮盖地图和其他UI元素

- [ ] **指南针显示**
  - 低功耗模式下，指南针是否正确显示
  - 转动设备，观察指南针箭头是否指向北方
  - 预期：指南针平滑旋转，北方标记为绿色

- [ ] **速度显示**
  - 开始移动（步行/驾车）
  - 观察速度数值是否实时更新
  - 预期：速度以km/h为单位显示，数字不跳动（monospaced）

- [ ] **已绘制像素数**
  - 观察已绘制像素数是否正确显示
  - 绘制几个像素后，数值是否增加
  - 预期：实时更新，与实际绘制数量一致

- [ ] **剩余点数显示**
  - 观察剩余点数是否正确显示
  - 绘制后点数是否减少
  - 预期：实时更新，与点数余额一致

---

### 3. 优化效果测试

- [ ] **屏幕亮度降低**
  - 进入低功耗模式后，屏幕明显变暗
  - 预期：亮度降至原来的40%左右（最低0.2）

- [ ] **屏幕防锁定**
  - 在低功耗模式下等待超过自动锁屏时间
  - 预期：屏幕不会自动锁定

- [ ] **地图渲染降级**
  - 观察低功耗模式下的地图
  - 预期：地图底图被隐藏（纯黑背景），但绘制预览层仍可见

- [ ] **批处理间隔增加**
  - 在低功耗模式下绘制像素
  - 观察控制台日志
  - 预期：批处理间隔从50ms增加到300ms

---

### 4. 交互监控测试

- [ ] **手势触发重置**
  - 进入GPS绘制模式
  - 等待4秒后，触摸屏幕
  - 观察是否重新开始5秒倒计时
  - 预期：不进入低功耗模式，倒计时重置

- [ ] **地图操作触发重置**
  - 进入GPS绘制模式
  - 平移、缩放、旋转地图
  - 观察每次操作后倒计时是否重置
  - 预期：持续操作不会进入低功耗模式

---

### 5. 状态恢复测试

- [ ] **退出低功耗后恢复**
  - 进入低功耗模式 → 退出
  - 观察屏幕亮度、地图UI是否恢复正常
  - 预期：完全恢复到进入前的状态

- [ ] **停止GPS绘制时清理**
  - 在低功耗模式下停止GPS绘制
  - 观察是否正确退出低功耗模式
  - 预期：亮度恢复、防锁定关闭、UI恢复

- [ ] **App进入后台**
  - 在低功耗模式下切换到其他app
  - 返回FunnyPixels app
  - 预期：GPS绘制继续，低功耗模式保持

---

### 6. 边界情况测试

- [ ] **GPS信号弱时**
  - 在GPS信号较弱的环境（室内）
  - 观察速度和指南针是否正常显示
  - 预期：速度显示0，指南针可能不旋转

- [ ] **点数耗尽时**
  - 绘制直到点数为0
  - 观察低功耗模式下的点数显示
  - 预期：显示"0 点"

- [ ] **快速进出低功耗模式**
  - 快速进入和退出多次（5秒进入 → 立即退出 → 5秒进入）
  - 预期：无崩溃，状态切换正常

- [ ] **首次使用标记持久化**
  - 卸载并重新安装app
  - 第一次进入低功耗模式
  - 预期：显示引导Toast

---

### 7. 性能测试（可选）

- [ ] **功耗测量**
  - 使用 Xcode Instruments → Energy Log
  - 测量正常GPS绘制模式的功耗
  - 测量低功耗GPS绘制模式的功耗
  - 预期：低功耗模式降低40-55%功耗

- [ ] **GPS精度测试**
  - 在低功耗模式下绘制一段路径
  - 对比正常模式绘制的路径
  - 预期：GPS精度保持不变，绘制质量一致

- [ ] **电池续航测试（长期）**
  - 完整充电后，开启GPS绘制30分钟
  - 记录正常模式的电量消耗
  - 重复测试低功耗模式的电量消耗
  - 预期：低功耗模式延长30%以上续航
